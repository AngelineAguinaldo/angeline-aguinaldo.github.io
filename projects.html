Projects

DisEasy

DisEasy is a web application designed to provide disease likelihood predictions and easy-to-interpret visualizations to its users. This incorporates learned models that are trained on features such as clinical laboratory results, demographics, and body mass index. Expected completion: June 2017.

Cell Mitosis Detection

This research project involves designing effective image analysis algorithms for detecting mitotic events in various cell types (i.e. embryonic, T-cells). These algorithms are applied on several image modalities (2D and 3D) including: phase contrast, fluorescence, and nuclear imaging. The automatic detection of mitotic events facilitate the automatic building of cell lineage trees which can be used to observe cell culture trends. Publication pending.

Mitochondria Dynamics in Aggressive Cancer Cells

This research project uses automated image segmentation and tracking algorithms to measure features of mitochondria dynamics in prostate cancer cells. This resulted in key inferences about the role of mitochondria in metastatic cancer cells.

Mitochondria Fission/Fusion Image Analysis Technique

This research project uses automated image segmentation and tracking algorithms, as well as, a self-identified feature to detect fission and fusion events in mitochondria organelles. This results in observable trends between cell oleic acid levels and the number of fusion events detected.

A Review of Generalized Benford's Law for JPEG Coefficients

This project re-produces the results obtained by Fu et. al. in the SPIE 2007 publication "A generalized Benford's law for JPEG coefficients and its applications in image forensics". This project elaborates on the application of Benford's law in identifying JPEG compression and other image properties.

This project uses computer vision concepts (i.e. fiducial matching, affine transformations, segmentation) to classify a sample fingerprint among a provided database of fingerprints.

Gain Engineering in Quantum Cascade Lasers

This research project identifies a mechanism that can be used to increase or decrease optical gain by-design in quantum cascade lasers. This allows for more control over the power efficiency of the optical device.

Compression Assistor

The Compression Assistor is a device that measures the depth of CPR compressions and supplies the first-responder with real-time feedback. This allows the health care provider to adjust compression depth accordingly and provide optimal, live-saving CPR compressions.
