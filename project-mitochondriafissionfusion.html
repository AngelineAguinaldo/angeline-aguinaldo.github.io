Mitochondria Fission/Fusion Image Analysis Technique

Mitochondrial organelles of cells with varying oleic acid concentrations have been captured in fluorescence microscopy images. The varying levels of oleic acid are expected to influence how these mitochondria function[2].

To understand the effects of various levels of oleic acid on the cell, the fission and fusion behavior of the mitochondria were observed. This was done by calculating the number of necks in each frame of a mouse cell-line image sequence.

Based on these observations, it was found that the fusion behavior of mitochondria is affected by the levels of oleic acid present in the mitochondria. Mitochondria exposed to high levels of oleic acid showed slower fusion rates than that of the wild-type mitochondria. Mitochondria starved of oleic acid showed faster fusion rates than that of the mitochondria.

Background

It has been found that mitochondrial fission and fusion behavior is indicative of mitochondria related cell death and maintenance of mitochondria DNA integrity[1], [5]. Therefore, the objective of this project was to observe the rate of fusion and fission of mitochondria organelles when exposed to various levels of oleic acid.

This technique was applied to mouse cell-line microscopy images from the National Institute of Health. A sample image can seen in the figure to the right. The cells in each oleic acid category were exposed to a specific level of oleic acid prior to the imaging. The oleic acid levels varied between three categories: high oleic acid, normal oleic acid, and no oleic acid. These cells were then imaged a frame per minute for 25 minutes. 42 unique cells were imaged after a specific oleic acid level exposure.

Methods

The parameter used to define fission and fusion was the necks that formed during the separation and union of the mitochondria. A neck is defined as the narrowest region between two large adjacent regions. The neck is circled in red on the figure to the left.

Segmentation

To segment the mitochondria, a filtering algorithm was used to extract the foreground from the background of the original image found in (a) in the figure below. In the provided dataset, the mitochondria channel was located on channel 4. This channel was extracted and a contrast filter was applied[4]. The contrast filter consisted of an iterative Gaussian low pass filter on the image data for 25 iterations as shown in (b).The contrast was adjusted as seen in c[3]. Following this, an Otsu threshold was determined and used to segregate the RGB image and convert it to a binary image as seen in (d). The small objects of pixel size less than 25 were removed, (e), to reduce noise. The image was then skeletonized to minimize incorrect connectivity between objects that may have been caused by halos. This can be found in (f).

The segmentation algorithm used in the provided data set

After the image was filtered, the boundaries on the binary image were located and locations of the boundary pixels were stored. Lines were then plotted on top of the figure at the boundary pixel locations. The colors of the plots were determined by the index of the filename found in the given file directory. The various mitochondria outlines can be found in the figure below.

Counting Necks

To count the necks, first each component was identified, and the boundary pixel locations were collected. The Euclidean distance between each pair of pixels was calculated. A 5x5 Gaussian filter was applied to the pair distance matrix to reduce noise. The minimum pixel pair distance within an individual object was determined. The number of pairs that matched that minimum pair distance determined the number of necks in that object. This was repeated and summed for each object. The code can be found below.

A 5x5 Gaussian filter was applied to the pair distance matrix to reduce noise. The minimum pixel pair distance within an individual object was determined. The number of pairs that matched that minimum pair distance determined the number of necks in that object. This was repeated and summed for each object. The code can be found in Appendix A.

This was inspired by a technique used by Westrate et al the 2014 publication found in the Public Library of Science[6].

function [totalnecks, neckLoc] = neckCount(B)
 
% B is a structure that stores the boundary pixel locations for each object.

totalnecks = 0;
neckLoc = [];
 
    for j = 1:numel(B)
        perim = [B{j}];
        pairdist = pdist(perim,'euclidean');
        
        kernel = [5 5]; % get size of image
        sigma = 0.5; % define standard deviation of gaussian distribution
        h = fspecial('gaussian',kernel, sigma); % create gaussian LP filt
        filterPerim = imfilter(pairdist,h,'symmetric');
        
        localmin = min(filterPerim,[],2);
        A = find(filterPerim == localmin);
        
        neck = numel(A);
        totalnecks = totalnecks + neck; % total branch regions
    end
end
MATLAB code used to count the number of mitochondria fission and fusion necks

Counting Objects

The number of objects were counted using MATLAB function bwlabel. This counted the number of separate objects found in each frame for each image sequence.

Results

Based on the colored segmentations found in the figure aforementioned, the red and magenta are for cells exposed to high oleic acid levels. The green and yellow cells were not exposed to any abnormal levels of oleic acid. The blue and cyan cells were starved of oleic acid.

The figure below shows a sample plot that corresponds to the number of necks found in one image sequence after 25 frames. The oscillatory behavior found in the plots is still under investigation. It may be attributed to the cycle of fission and fusion that occurs in an attempt to mix mitochondria contents[1]. Additionally, this could be the cause of noise because of the microscopy technique.

Conclusions and Discussion

In addition to the number of necks, the number of objects in each frame was collected. Based on the neck and object relationship, it can be inferred that fusion is occurring. It can be said that the rate of fusion decreases as the oleic acid level increases.

Ultimately, this implies that mitochondria exposed to high oleic acid levels have a slower rate of fusion than that of wild type mitochondria. And the mitochondria starved of oleic acid experience have a faster rate of fusion that that of wild type mitochondria.

A potential explanation for this could be that the starved mitochondria fuse together faster than the wild type in an attempt to complement and re-encode their damaged DNA and the mitochondria with high oleic acid do not need to re-encode their DNA as quickly because they do not damage as quickly.

Future Work

Several items related to this project are still under investigation. The intent is to complete validation studies on the accuracy of the neck count being reported. This can be will be done by circling the necks being detected by the algorithm and noting the correctness. Statistical studies will also be done to compare the sample observations to the total population. Additionally, the algorithm will be modified to adaptively crop the image to the region of interest.

Project Details

This project was completed within a ten week graduate-level course entitled, Cell Tissue Image Analysis.
